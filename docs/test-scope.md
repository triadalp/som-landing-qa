# Test Scope

## In Scope

The following areas will be tested:

### Functional Testing
- Contact form
- Call buttons
- Telegram links
- Navigation links
- External links
- Form validation

### UI Testing
- Layout consistency
- Text visibility
- Button states
- Responsive behavior
- Section spacing
- Images and icons

### Responsive Testing
- Mobile devices
- Tablet devices
- Desktop resolution

### Basic SEO Checks
- Page title
- Meta description
- Heading structure
- Language attribute

### Analytics Verification
- CTA button events
- Form submission events
- Section view events

---

## Out of Scope

The following areas will NOT be tested:

- Backend performance testing
- Security penetration testing
- Load testing
- Database testing
- Browser support for outdated browsers
- Automated testing

---

## High Priority Areas

Critical areas for business:
- Contact form
- Phone call buttons
- Contact information
- Mobile usability

---

## Low Priority Areas

Lower priority areas:
- Minor animation issues
- Small visual inconsistencies
- Non-critical spacing issues
